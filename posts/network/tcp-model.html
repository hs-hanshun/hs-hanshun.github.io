TCP/IP网络模型有哪几层 #

根据OSI(Open System Interconnection Reference Model)定义的网络模型，网络一共分为7层，分别是应用层、表示层、会话层、传输层、网络层、数据链路层、物理层。OSI的七层体系结构概念非常清除，理论也很完整，但是它比较复杂而且不使用。所以现实中，我们经常用的是五层网络模型，分别是应用层、传输层、网络层、数据链路层、物理层。
